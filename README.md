# Integrated-Banking-Risk-Analytics-System
ML-powered banking analytics platform for loan prediction, fraud detection, and customer segmentation.

## Overview
An end-to-end banking analytics platform that integrates Loan Risk Prediction, Fraud Detection, and Customer Segmentation using Machine Learning techniques.

## Features

### Loan Risk Prediction
Predicts whether a loan application is likely to be approved based on applicant details such as income, loan amount, and CIBIL score.

### Fraud Detection
Identifies potentially fraudulent banking transactions using machine learning models.

### Customer Segmentation
Groups customers into different segments using K-Means clustering for targeted financial services and business insights.
